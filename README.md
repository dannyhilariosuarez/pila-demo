# pila-demo

A simple Express API used for demonstrating [pila](https://github.com/dannyhilariosuarez/pila) — an AI agent orchestrator.

## Setup

```bash
npm install
npm run build
npm start
```

## API Endpoints

- `GET /items` — List all items
- `POST /items` — Create a new item
- `GET /items/:id` — Get an item by ID

## Development

```bash
npm run dev    # Run with ts-node
npm test       # Run tests
npm run lint   # Type-check
```
